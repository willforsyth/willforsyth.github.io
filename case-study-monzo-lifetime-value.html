Monzo | Lifetime value

I wanted to do a personal project looking at how we could build on the information that the app gathers and provides to increase business growth and add value to the customers.

Personal project

"How could we increase revenue, add value and increase the LTV and use of the app."

Business gaols

"How can we reduce the need for users to go to shops during the covid epidemic, and save them money.""

Brainstorm and sketch

Allow a detailed view into spending habits and budgets, to save money, give more control and reduce carbon footprints.

    Ideas
  • Create shopping lists
  • Gather data on shopping habbits
  • Receipt breakdown
  • Order and compare prices from other stores
    • Shop cheap
    • Show eco friendly
  • View spending habits

Receipt breakdown

Allow the user to click on a receipt and view a breakdown of all the purchased items

Order again

Allow the user to click on the receipt and place the same order again. Using checkboxes to update the order and then picking a pot to pay from.
